# 《我老公是条龙》角色四视图需求表

适用范围：

- 第一季前期核心角色统一
- 当前优先角色：敖玄、沈映晚、赵元启
- 用途：角色定妆、后续镜头一致性、剧情出图参考

---

## 统一标准

所有角色统一按“四视图”执行：

1. 正面全身
2. 左侧面全身
3. 背面全身
4. 正面头部特写

说明：

- 不额外扩展 3/4 视图
- 不额外扩展侧面头部特写
- 先保证角色稳定，再进入动作图和剧情镜头

---

## 一、敖玄四视图需求

### A. 人形基础四视图

必须产出：

1. 正面全身
2. 左侧面全身
3. 背面全身
4. 正面头部特写

统一要求：

- 成年男性，约25-30岁感
- 身形修长挺拔，约185cm视觉比例
- 银白长发，长度到腰际
- 赤红色眼睛
- 面容冷峻、骨相分明、五官英俊但克制
- 身穿白色长袍，质感轻盈，有隐约龙族尊贵感
- 靴子为白色或浅银白
- 整体气质：疏冷、强大、压迫感中带克制

不能跑偏：

- 不能画成少年感太强
- 不能画成阴柔妖艳系
- 不能有夸张珠宝配饰
- 不能把发色画成纯灰或金白

### B. 正面头部特写重点

重点统一：

1. 赤眸必须稳定
2. 银发层次要清楚
3. 鼻梁高，眼神深，唇色偏淡
4. 脸型偏冷峻，不要幼态化
5. 发际线和额头比例要固定

### C. 白蛇形态参考

必须产出：

1. 白蛇盘卧版
2. 白蛇抬头版
3. 白蛇受伤版
4. 白蛇腾空版

统一要求：

- 通体雪白
- 眼睛赤红
- 鳞片细密有月光感
- 体型不是粗壮蟒蛇，更偏灵性细长
- 气质不是普通蛇，是高贵且危险的龙族投影

### D. 真龙形态预设

虽然前10集暂未正式使用，但要提前锁：

1. 百丈白龙正面参考
2. 白龙侧飞参考
3. 白龙怒战参考

统一要求：

- 龙鳞纯白偏银
- 龙眼赤红
- 龙角修长锋利
- 气质要神圣与危险并存

## 二、沈映晚四视图需求

### A. 基础四视图

必须产出：

1. 正面全身
2. 左侧面全身
3. 背面全身
4. 正面头部特写

统一要求：

- 成年女性，18-22岁感
- 身高约165cm视觉比例
- 身形纤细，不弱不病
- 黑发，简单发髻，青色发带
- 眼睛清亮，神情温柔
- 穿素青衣裙，带浅色花纹，不华贵但干净
- 布鞋，采药女气质
- 整体气质：温软、清秀、灵气、胆子大

不能跑偏：

- 不要画成过度富贵大小姐
- 不要画成妖艳妆容
- 不要发型太复杂

### B. 正面头部特写重点

重点统一：

1. 眼睛清亮，不要画钝
2. 面部线条柔和但不幼态
3. 发髻和发带位置要稳定
4. 唇形自然，不要浓妆感
5. 神情以温柔和灵气为主

### C. 常用状态需求

必须产出：

1. 背药篓采药版
2. 居家照顾版
3. 抱蛇版
4. 门槛坐姿版

## 五、执行提醒

四视图不是为了好看，是为了后续所有镜头统一。

因此：

1. 先锁主角，不要一开始铺太多角色
2. 头部特写必须和全身版保持同一张脸
3. 先锁基础版，再做动作版
4. 每次出图后保留“最终选定参考图”
5. 后续所有镜头词都要以选定参考图为锚点
