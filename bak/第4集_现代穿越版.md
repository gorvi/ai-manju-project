# 《我老公是条龙》第4集《穿越》现代→古代穿越版

**核心设定：**
沈映晚是现代都市女性，某天在上海外白渡桥离奇穿越到古代，救下了正在渡劫的敖玄。这个第4集是整部剧的"穿越起点"，也是核心钩子。

**穿越逻辑：**
敖玄在人间历劫时与沈映晚命中注定相连——当他渡劫失败濒死时，沈映晚从另一个世界被召唤而来，成为唯一能救他的人。两人在前世（现代）已有羁绊，但敖玄渡劫后会忘记这段记忆，直到慢慢重新爱上她。

## 穿越线索埋点

1. **第4-8场**：美男亲吻时，嘴唇触感像"很久以前认识的人"
2. **第4-9场**：那一巴掌和白衣男子消散的眼神——他似乎早就预料到
3. **第4-12场**：时光隧道中闪过模糊画面——暗示前世记忆
4. **第4-20场**：白蛇的赤红瞳孔和美男一模一样，呼应"我的老公是条龙"
5. **第4-22场**：沈映晚说"和桥上的一模一样"——她自己也不明白为什么

## 集尾悬念

沈映晚低头看着怀里奄奄一息的白蛇，忽然想起了什么，脱口而出：
台词（字幕）："你……是不是在等我来救你？"

白蛇在她怀里微微动了动，赤红的眼眸中似乎闪过一丝波动。

黑场字幕："她来自一千年后的上海，而他在那里等了她三百年。"
