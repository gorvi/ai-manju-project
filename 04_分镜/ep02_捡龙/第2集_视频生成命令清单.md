## 命令格式模板

```bash
dreamina multimodal2video \
  --image <场景底图16:9.png> \
  --image <角色参考图9:16.png> \
  --prompt "中文镜头画面描述" \
  --model_version seedance2.0 \
  --ratio 9:16 \
  --duration N \
  --poll 60
```
